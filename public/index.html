About Us

O IWannaBeNotified é um serviço de notificação de páginas da internet disponibilizado por meio de uma API REST, que permite monitorar sites por atualizações e receba uma notificação sempre que o site tiver uma nova atualização. Veja as opções para receber sua notificação:

O serviço é customizável a ponto de você configurar quais informações você deseja extrair do site, o template que você deseja receber a notificação e até mesmo os canais que você deseja ser notificado.

  • Email
  • Telegram
  • Web Hooks
  • Websocket
  • SMS (Em breve)
  • Facebook Messenger (Em breve)
  • Whatsapp (Em breve)
  • Web Push (Em breve)

O serviço permite que você monitore o conteúdo de uma página Web apenas informando a Url dessa página e sempre o serviço identificar uma mudança de conteúdo. Normalmente as páginas da web mudam com muita frequência, algumas delas, chegam a ser diferente a cada acesso (propagandas e datas)

Além de receber simples notificações é possível receber o conteúdo extraído da página. Caso você conheça algum pouco de Javascript, é possível executar scripts para extração e manipulação do conteúdo da página antes da obtenção das informações, desta maneira você pode extrair apenas uma parte da página se desejar.

A verificação do site é feita de uma forma cíclica, ou seja, de tempos em tempos o serviço acesse o site desejado para obter o conteúdo dele, é possível configurar este tempo de verificação. Assim, você pode receber o conteúdo sempre que esse ciclo ocorra, dependendo do seu propósito.

Também é possível configurar o monitoramento para que você receba apenas conteúdos quando forem diferentes do conteúdo encontrado anteriormente (change) ou até mesmo conteúdos únicos.

Frequently Asked Questions

Existe algum tipo de authenticação na API do IWannaBeNotified?

A API permite efetuar requisições de duas maneiras, a primeira é por meio de uma autenticação prévia. Para facilitar sua vida, é possivel efetuar requisições sem se autenticação, de uma maneira anônima.

Existe alguma ducumentaçao para integração do serviço?

A documentação da API pode ser acessada no Swagger da aplicação ou no link a seguir: (Documentação IWannaBeNotified API).

Como posso testar os meu Scripts?

Existe um endpoint que permite você efetuar teste do script de uma forma mais rápida e facil.

A API esta indisponivel, o que eu faço?

Envie um e-mail notificando o problema para achimid@hotmail.com que o seu problema será analisado o mais rápido possivel.

Existe alguma maneira de contribuir com a iniciativa?

Atualmente nossos serviços não é rentável, estamos aceitando doações de qualquer tipo. Mas caso você não possa realizar uma doação, estamos procurando pessoas para auxiliar no desenvolvimento do serviço. Mas no final de tudo, a sua divulgação é o mais importante para nos no momento.
